# HG6145F1 Password Generation GUI tool
Admin password generation tool GUI for fiberhome Router (AT)<br>
Before use you need to have python installed on you system! use the link below to download Python <br>
<a href="https://www.python.org/ftp/python/3.13.11/python-3.13.11-amd64.exe">Download Python Version 3.13.11 For Win64</a> <br>
Open the app, <br> clic on <b>"gen"</b> button then paste your MAC address in the dialog box that appear <br>
TADA! Enjoy
